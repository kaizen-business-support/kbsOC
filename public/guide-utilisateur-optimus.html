🎯 1. Introduction

Optimus est un système d'analyse financière avancée conçu spécifiquement pour les analystes financiers travaillant dans l'environnement réglementaire de la BCEAO (Banque Centrale des États de l'Afrique de l'Ouest).

✨ Fonctionnalités Principales

  • Analyse Multi-Années : Analyse comparative sur plusieurs exercices
  • Conformité BCEAO : Vérification automatique des normes réglementaires
  • Recommandations Sectorielles : Conseils adaptés au secteur d'activité
  • Génération de Rapports PDF : Documents professionnels automatisés
  • Tableau de Bord Interactif : Visualisations graphiques avancées

📊 2. Saisie des Données Financières

2.1 Configuration Initiale

Avant de commencer l'analyse, vous devez configurer les informations de base :

  1. Accédez à l'URL : http://localhost:3001/optimus
  2. Dans la section "Saisie des Données", renseignez :
    • Nom de l'entreprise
    • Secteur d'activité (Industrie, Commerce, Services, Agriculture)
    • Année de référence
    • Période d'analyse (nombre d'années)

Le secteur choisi influence les normes BCEAO et les recommandations générées.

2.2 Import par Fichier Excel

  1. Préparez vos fichiers Excel avec les états financiers
  2. Accédez à "Saisie des Données"
  3. Cliquez sur "Importer Excel"
  4. Sélectionnez vos fichiers (un par année)
  5. Vérifiez la correspondance des colonnes
  6. Validez l'import

2.3 Structure des Données Requises

📈 Bilan

  • Actif Immobilisé
  • Actif Circulant
  • Trésorerie Actif

🔍 3. Analyse Financière Détaillée

L'analyse détaillée se compose de 5 onglets principaux :

3.1 États Financiers

Affiche les données financières avec :

  • Données multi-années : Comparaison année par année
  • Variation annuelle : Pourcentage d'évolution sous chaque valeur
  • Tendance : Direction générale (Amélioration/Dégradation/Stable)
  • Croissance moyenne : Progression moyenne sur la période

3.2 Ratios Détaillés

Calculs automatiques des ratios financiers :

  • ROE : Rentabilité des capitaux propres
  • ROA : Rentabilité de l'actif
  • Ratio de Liquidité : Capacité de paiement à court terme
  • Ratio d'Endettement : Niveau d'endettement

Chaque ratio inclut sa formule de calcul et son interprétation.

3.3 Évolution & Tendances

Analyse approfondie des variations :

  • Graphiques d'évolution : Visualisation des tendances
  • Analyse des écarts : Identification des variations significatives
  • Points forts et faibles : Synthèse automatique

3.4 Conformité BCEAO

Vérification réglementaire complète :

  • Normes sectorielles : Adaptation aux spécificités du secteur
  • Historique de conformité : Évolution sur plusieurs années
  • Recommandations personnalisées : Actions concrètes par ratio
  • Formules des ratios : Calculs détaillés et explications

3.5 Recommandations

Plan d'action stratégique :

  • Points forts : Leviers de performance identifiés
  • Axes d'amélioration : Domaines nécessitant une attention
  • Actions court terme : Mesures immédiates (3-6 mois)
  • Actions moyen terme : Stratégie à long terme (6-12 mois)

🏛️ 4. Module Conformité BCEAO

4.1 Utilisation du Module

  1. Accédez à l'onglet "Conformité BCEAO"
  2. Consultez le score global de conformité
  3. Explorez les catégories : Liquidité, Solvabilité, Rentabilité
  4. Cliquez sur l'icône "Action" pour voir les détails d'un ratio
  5. Consultez la formule et les recommandations spécifiques

4.2 Interprétation des Statuts

  • 🟢 Excellent/Bon : Conforme aux normes, situation saine
  • 🟡 Acceptable : Légèrement sous les normes, surveillance nécessaire
  • 🟠 Insuffisant : Sous les normes, actions correctives requises
  • 🔴 Critique : Très sous les normes, intervention urgente

4.3 Tendances Multi-Années

Le module affiche l'évolution de chaque ratio sur toute la période d'analyse :

  • 🔼 Amélioration : Ratio en progression
  • 🔽 Dégradation : Ratio en déclin
  • ➡️ Stable : Ratio constant

🧮 6. Formules des Ratios Financiers

6.1 Ratios de Liquidité

Ratio de Liquidité Générale

Actif Circulant / Passif Circulant

Mesure la capacité de l'entreprise à honorer ses dettes à court terme avec ses actifs liquides. Norme BCEAO : ≥ 1,2

Ratio de Liquidité Réduite

(Actif Circulant - Stocks) / Passif Circulant

Évalue la liquidité en excluant les stocks, moins facilement convertibles. Norme BCEAO : ≥ 1,0

Ratio de Liquidité Immédiate

Disponibilités / Passif Circulant

Mesure la capacité de paiement immédiat avec les liquidités disponibles. Norme BCEAO : ≥ 0,3

6.2 Ratios de Solvabilité

Ratio d'Autonomie Financière

(Capitaux Propres / Total Bilan) × 100

Indique le degré d'indépendance financière vis-à-vis des créanciers. Norme BCEAO : ≥ 20%

Ratio d'Endettement

(Total Dettes / Total Bilan) × 100

Mesure le niveau d'endettement global par rapport aux actifs. Norme BCEAO : ≤ 70%

6.3 Ratios de Rentabilité

Rentabilité des Capitaux Propres (ROE)

(Résultat Net / Capitaux Propres) × 100

Mesure la rentabilité des capitaux investis par les actionnaires. Norme BCEAO : ≥ 10%

Rentabilité de l'Actif (ROA)

(Résultat Net / Total Actif) × 100

Indique l'efficacité dans l'utilisation des actifs pour générer des profits. Norme BCEAO : ≥ 5%

6.4 Ratios d'Activité

Rotation de l'Actif

Chiffre d'Affaires / Total Actif

Indique l'efficacité d'utilisation des actifs pour générer du CA. Norme BCEAO : ≥ 0,8

🔧 7. Résolution de Problèmes

7.1 Problèmes Courants

❌ Erreur d'Import Excel

Symptôme : "Format de fichier non reconnu"

Solution :

  • Vérifiez que le fichier est au format .xlsx ou .xls
  • Assurez-vous que les colonnes ont des en-têtes
  • Vérifiez qu'il n'y a pas de cellules fusionnées

❌ Données Manquantes

Symptôme : "Données incomplètes pour l'analyse"

Solution :

  • Vérifiez que tous les champs obligatoires sont renseignés
  • Contrôlez la cohérence des montants (Actif = Passif)
  • Assurez-vous que les valeurs sont numériques

❌ Rapport PDF Vide

Symptôme : Le PDF généré est vide ou incomplet

Solution :

  • Attendez la fin complète de la génération
  • Vérifiez que l'analyse a été effectuée avant génération
  • Rechargez la page et recommencez

7.2 Support Technique

En cas de problème persistant :

  • Consultez les logs du navigateur (F12)
  • Vérifiez votre connexion internet
  • Essayez un autre navigateur
  • Contactez l'administrateur système

7.3 Bonnes Pratiques

  • Sauvegarde : Exportez régulièrement vos analyses
  • Validation : Vérifiez la cohérence des données saisies
  • Documentation : Conservez les sources de vos données
  • Mise à jour : Utilisez toujours la dernière version
